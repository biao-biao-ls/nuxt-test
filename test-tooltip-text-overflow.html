测试用例

短文本： "James" + "Sales Manager" - 正常显示
中等长度： "test_01" + "Business Development Representative" - 职位截断
长姓名： "Alexander Johnson" + "Senior Software Development Engineer" - 都可能截断
超长文本： "Christina Rodriguez Martinez" + "International Business Development Manager" - 明显截断

文本截断优化

  • 设置 max-width: 250px 限制 tooltip 最大宽度
  • 使用 overflow: hidden 隐藏超出内容
  • 使用 text-overflow: ellipsis 显示省略号
  • 使用 white-space: nowrap 防止文本换行
  • 使用 flex: 1; min-width: 0 确保文本容器能够收缩
  • 保持头像固定尺寸，只截断文本部分
  • 姓名和职位都支持独立截断
测试说明：

鼠标悬停在左右两侧的客服头像上，对比 tooltip 的文本处理效果。 左侧会显示完整文本（可能很宽），右侧会截断过长的文本并显示省略号。 注意观察不同长度文本的处理效果。
